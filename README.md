# Lambda Playwright

## 概要
AWS Lambda上でPlaywrightを実行するためのスクリプトを配置・管理するプロジェクトです。

## 利用技術
- Node.js（バージョン管理はVoltaで実施）
- Playwright
- TypeScript
- AWS Lambda
- Volta
- pnpm

## 想定利用者
- 最低限のNode.js知識がある方

## セットアップ手順
1. Voltaをインストールし、Node.jsのバージョン管理を行う
2. pnpmをインストール
3. 依存パッケージのインストール
	```sh
	pnpm install
	```
